Despre Aplicatie

Board Games Hub este o aplicatie web pentru grupuri de prieteni care se intalnesc periodic pentru seri de board games. Aplicatia rezolva o problema simpla si concreta: ce joc jucam vineri seara?

Membrii grupului pot vota jocul saptamanii, pot vedea istoricul partidelor trecute si pot urmari cine este campionul grupului la Catan, Sah sau orice alt board game.
Nu mai e nevoie de foi de hartie sau mesaje in grup, totul este centralizat intr-un singur loc.

Planul de dezvoltare

Pasi de implementare

Cont de utilizator, votare joc saptamanal, inregistrare scoruri dupa partida, clasament grup.
